# مفتاح السعادة — Miftah Saada 🛒🌟

**مفتاح السعادة** هو متجر إلكتروني متكامل واحترافي لبيع الهواتف الذكية، الإلكترونيات، والإكسسوارات. تم بناء المشروع باستخدام **HTML / CSS / JavaScript (Vanilla)** للواجهة الأمامية، مع الاعتماد الكامل على خدمات **Supabase Cloud** كخلفية (Backend) وقاعدة بيانات سحابية.

---

## ✨ المميزات الرئيسية

### 🛍️ واجهة المتجر (Storefront)
- 🎨 **تصميم عصري وجذاب (Premium Design):** واجهة مستخدم حديثة ومتجاوبة 100% مع جميع أحجام الشاشات.
- 🌙 **الوضع الليلي والنهاري (Dark/Light Mode):** دعم كامل للتبديل بين الأوضاع مع حفظ تفضيلات المستخدم.
- 🛒 **نظام سلة متطور:** سلة مشتريات ديناميكية مع تحديث فوري للكميات والأسعار.
- 🔍 **تصفح مرن:** إمكانية تصفح المنتجات حسب الفئات والماركات مع صفحات مخصصة لكل منتج.
- 📱 **تجربة مستخدم تفاعلية:** إشعارات منبثقة (Toasts)، رسوم متحركة سلسة (Micro-animations)، وتجربة تصفح سريعة.

### 🛡️ لوحة تحكم الإدارة (Admin Dashboard)
- 📊 **إحصائيات وتقارير المبيعات:** لوحة رئيسية تعرض إجمالي المبيعات، الطلبات، المنتجات الأكثر مبيعاً مع رسوم بيانية (Chart.js).
- 📦 **إدارة المنتجات:** إضافة، تعديل، وحذف المنتجات بسهولة مع رفع الصور مباشرة إلى Supabase Storage.
- 🏷️ **إدارة الفئات والماركات:** تنظيم المنتجات ضمن فئات وماركات مع إمكانية إضافة "الفئات/الماركات الشهيرة تلقائياً".
- 📋 **إدارة الطلبيات والعملاء:** متابعة حالة الطلبات (معلق، مكتمل، ملغي)، طباعة الفواتير، وتصدير البيانات.
- ⚙️ **إعدادات الموقع المتقدمة:** 
  - تخصيص ألوان الموقع (استخراج ذكي للألوان من الشعار).
  - تعديل نصوص وعناوين الواجهة الرئيسية (Hero Section).
  - إدارة روابط الشبكات الاجتماعية ومعلومات التواصل وتحديثها لحظياً في المتجر.
- 🔒 **حماية وأمان:** نظام مصادقة (Authentication) متين يمنع دخول غير المصرح لهم للوحة التحكم.

## 🛠️ التقنيات المستخدمة

| التقنية | الدور والاستخدام |
|---------|-----------|
| **HTML5 / CSS3** | بناء الهيكل الأساسي وتصميم واجهات عصرية (Glassmorphism، تأثيرات Hover). |
| **Vanilla JavaScript** | المنطق التفاعلي، إدارة حالة التطبيق (State)، والتواصل مع قواعد البيانات. |

| **Google Fonts** | خط `Tajawal` لدعم ممتاز للغة العربية. |

## ⚙️ إعداد وتشغيل المشروع

### 1. إعداد Supabase
1. قم بإنشاء حساب ومشروع جديد على [Supabase](https://supabase.com).
2. انسخ `Project URL` و `anon public key`.
3. الصق القيم في ملف `js/supabase.js`:
   ```javascript
   const supabaseUrl = 'YOUR_SUPABASE_URL';
   const supabaseAnonKey = 'YOUR_SUPABASE_ANON_KEY';
   ```

### 2. إعداد قاعدة البيانات (PostgreSQL)
تحتاج إلى إنشاء الجداول التالية في Supabase:
- `products`: (id, name, price, old_price, category, brand, description, stock, specs, media_urls, created_at)
- `categories`: (id, name, slug)
- `brands`: (id, name)
- `orders`: (id, customer_name, phone, address, items, total_price, status, created_at)
- `messages`: (id, name, email, subject, message, created_at)
- `site_settings`: (id, hero_title, hero_subtitle, primary_color, secondary_color, logo_url, ...)

*(يمكنك استخدام لوحة التحكم لإضافة البيانات الأولية كالفئات والمنتجات التجريبية)*

### 3. إعداد التخزين (Storage)
- قم بإنشاء Bucket باسم `products` لرفع صور المنتجات.
- قم بإنشاء Bucket باسم `site-assets` لرفع الشعارات وصور الخلفية.
- تأكد من جعل الـ Buckets **عامة (Public)** ليتم عرض الصور في المتجر.

### 4. التشغيل المحلي
المشروع لا يتطلب خادم خلفي (Node.js/PHP) معقد بفضل Supabase.
- افتح المشروع في محرر النصوص (مثل VSCode).
- استخدم إضافة **Live Server** لتشغيل الموقع محلياً ومعاينة التغييرات الحية.
