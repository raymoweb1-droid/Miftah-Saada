# مفتاح السعادة — Miftah Saada 🛒

متجر إلكتروني متكامل للهواتف الذكية والإلكترونيات، مبني بـ **HTML / CSS / JavaScript** وبقاعدة بيانات **Supabase** السحابية.

---

## 🌟 مميزات المشروع

- 🎨 **تصميم عصري** بألوان ديناميكية قابلة للتخصيص من لوحة التحكم
- 📱 **متجاوب بالكامل** مع جميع أحجام الشاشات (موبايل، تابلت، ديسكتاب)
- 🛒 **سلة تسوق** كاملة مع إدارة الكميات
- 🔐 **لوحة تحكم Admin** لإدارة المنتجات والطلبات والإعدادات
- ☁️ **Supabase Cloud** لقاعدة البيانات والصور والإعدادات
- 🔔 **شريط إعلانات** متحرك قابل للتفعيل/التعطيل
- 🗺️ **خريطة تفاعلية** في صفحة التواصل

---

## 📁 هيكلة المشروع

```
مفتاح السعادة/
├── index.html          # الصفحة الرئيسية
├── shop.html           # صفحة المتجر
├── cart.html           # سلة المشتريات
├── checkout.html       # إتمام الطلب
├── product.html        # تفاصيل المنتج
├── contact.html        # تواصل معنا
├── login.html          # تسجيل الدخول
├── register.html       # إنشاء حساب
├── admin.html          # لوحة التحكم
├── css/
│   └── style.css       # ملف CSS الموحد
├── js/
│   ├── main.js         # المنطق الرئيسي للموقع
│   └── supabase.js     # إعداد Supabase
└── assets/             # الصور والأيقونات
```

---

## 🚀 كيفية التشغيل

1. استنسخ المستودع:
   ```bash
   git clone https://github.com/YOUR_USERNAME/miftah-saada.git
   ```

2. افتح ملف `index.html` مباشرة في المتصفح، أو استخدم خادم محلي مثل **Live Server** في VSCode.

3. لتفعيل لوحة التحكم، تأكد من إعداد Supabase وإدخال بيانات الجداول المطلوبة.

---

## ⚙️ الإعداد مع Supabase

- أنشئ مشروعاً جديداً على [supabase.com](https://supabase.com)
- أنشئ الجداول التالية: `products`, `orders`, `messages`, `site_settings`
- انسخ **URL** و**Anon Key** وضعهما في `js/supabase.js`
